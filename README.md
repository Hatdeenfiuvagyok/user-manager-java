# User Manager Console Application

Java console-based application for managing users.

## Features
- Load users from file
- Save users to file
- Add / Delete / Search users
- Menu-based interface
- Object-Oriented Design
- Interface usage

## Technologies
- Java
- File I/O
- Collections
- OOP

## Description
This project was created for learning purposes to practice
Java fundamentals, object-oriented programming, and file handling.
